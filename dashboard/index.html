Installation Instructions

Follow these steps to connect your Shopify store:

  1. Go to your Shopify Admin > Online Store > Themes
  2. Click Actions > Edit code
  3. Open theme.liquid file
  4. Paste the embed code below just before the closing </body> tag
  5. Click Save
  6. Visit your store home page once so sections and pages get auto-detected
